# 🚀 Destiminder

Destiminder is a smart location-based reminder mobile application built using Flutter.  
It helps users set destination alerts and get notified when they reach their selected location.

---

## 📱 Features

- 📍 Live Location Tracking
- 🗺 Map Picker for selecting destination
- 🔔 Smart Arrival Notifications
- 📳 Custom Vibration Alerts
- 🎙 Voice Alert Support
- 📜 Arrival History Tracking
- 🔄 Background Monitoring
- 🎨 Clean and Modern UI

---

## 🛠 Tech Stack

- Flutter
- Dart
- Flutter Maps API
- Android Native Services
- Background Location Services
